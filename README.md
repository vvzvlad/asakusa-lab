# Asakusa Lab - Керамическая студия

Одностраничный сайт для керамической студии Asakusa Lab в Москве.

## Структура проекта

```
.
├── index.html          # Основной HTML-файл
├── css/
│   └── style.css       # Стили сайта
├── js/
│   └── main.js         # JavaScript для навигации и интерактивности
├── IMG_0700 (1).PNG    # Логотип студии
└── README.md           # Этот файл
```

## Структура сайта

Сайт состоит из 7 полноэкранных секций:

1. **Hero (Главная)** - Логотип, описание, навигация
2. **О нас** - Информация о студии и сообществе
3. **Наше оборудование** - Подробное описание оборудования студии
4. **Где мы находимся** - Контакты и карта
5. **Цены и условия** - Прайс-лист услуг
6. **Заказать разработку** - Информация об услугах разработки
7. **Наше производство** - Каталог продукции с ссылками на маркетплейсы

## Функциональность

- **Full-page scroll**: плавный переход между секциями при прокрутке
- **Smooth scroll navigation**: быстрый переход по ссылкам
- **Keyboard navigation**: управление стрелками, PageUp/PageDown, Home/End
- **Touch support**: свайпы на мобильных устройствах
- **Responsive design**: адаптивная верстка для Desktop/Tablet/Mobile

## Дизайн

- Минималистичный стиль
- Черный фон (#000000)
- Фирменный цвет: #2D491F (зеленый)
- Современная типографика
- Плавные анимации и переходы

## Запуск локально

Для локального тестирования можно использовать любой веб-сервер:

```bash
# Python 3
python3 -m http.server 8000

# Затем открыть в браузере:
# http://localhost:8000
```
