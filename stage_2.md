# ТЗ на разработку сайта Asakusa Lab

## 1. Общая информация

**Проект:** Сайт керамической студии Asakusa Lab  
**Локация:** Москва  
**Формат:** Одностраничный прокручиваемый сайт (full-page scroll)  
**Дизайн:** Минималистичный, чёрный фон, современная эстетика

---

## 2. Структура навигации

Сайт состоит из **6 основных экранов**, каждый — полная высота окна браузера. Навигация работает двумя способами:

1. **Прокручивание страницы** (scroll) — переход между экранами последовательно
2. **Навигационные ссылки** — быстрый переход на нужный экран

Макет навигации:
- Экран 1: Логотип + tagline + навигационные ссылки
- Остальные экраны (2-6): Заголовок раздела + контент + при необходимости дополнительные ссылки внизу

## 5. Экран 3 — Наше оборудование

### Заголовок экрана:
"Наше оборудование"

### Контент:
Подробное описание всего оборудования студии, организованное по категориям:

#### Печи для обжига:
- **Большая печь:** 80 литров
- **Маленькая (быстрая) печь:** 10 литров
  - Время обжига: ~10 часов
  - Возможность проводить 2 обжига в день
  - Идеальна для тестирования и прототипирования
- **Контроллеры собственного производства:** позволяют программировать процесс обжига и отслеживать реальный график нагрева для отладки

#### Гончарный круг:
- 2 гончарных круга **iMold Professional V3** (последних версий)
- Характеристика: мощные, тихие, надежные профессиональные аппараты
- Выбор качества: студия отдает приоритет лучшему оборудованию

#### Глазури и их компоненты:
- Собственная база для производства глазурей
- Обширный запас компонентов для глазурей
- Обратный осмос (для подготовки воды для глазурей для отсуствия оксидов железа)

#### Оборудование для приготовления и хранения:
- Много места для хранения материалов
- Большая удобная раковина с ринзером для мытья баночек
- Фильтры-Воронки для переливания
- Мешалки (собственного производства)

#### Вытяжка и вентиляция:
- Вытяжка для печи (позволяет работать во время обжига без неприятных запахов)

#### Покрасочное оборудование:
- Покрасочная камера
- Несколько краскопультов
- Компрессор (большой, но тихий) (позволяет красить много изделий повторяемо)

#### Литьё в гипсовые формы (Slip casting):
- Большая мешалка для шликера (жидкая керамическая масса для литья)
- Стол для литья
- База для производства гипсовых форм
- Полный цикл: от создания формы до литья

#### 3D-печать и цифровое литьё:
- **Несколько 3D-принтеров** для:
  - Печати вспомогательных инструментов
  - Печати вырубок
  - Печати штампов
  - Печати индивидуальных маркировочных штампов для керамистов
  
- **Технология "Цифровое литьё"** (Digital Casting):
  - Процесс: 3D-модель → печать формы на 3D-принтере → создание силиконовой формы → тиражирование гипсовых форм → литьё шликера в формы
  - Позволяет производить неограниченное количество идентичных гипсовых форм на основе одной печатной формы
  - **Курсы по цифровому литью** проводятся в студии

#### Вспомогательные пространства:
- **Чистая зона:** помещение без пыли для работы за ноутбуком и отдыха на диване
- **Фотозона:** оборудована лайтбоксом для фотографирования готовых изделий
- **Выставочная зона:** небольшая выставочная галерея в коридоре, где керамисты могут выставить свои работы для демонстрации посетителям (без необходимости каждый раз доставать изделия)

## 10. Технические требования

### Адаптивность:
- **Полная адаптивная верстка** для всех устройств:
  - Desktop (от 1024px)
  - Tablet (768px - 1023px)
  - Mobile (до 768px)
  
### Функциональность:
- **Full-page scroll:** при прокручивании страницы экраны переходят плавно 
- **Якорные ссылки:** навигационные ссылки на первом экране должны вести на соответствующие экраны (с smooth scroll)
- **Стрелка вниз (на первом экране):** также должна быть кликабельна для прокрутки до следующего экрана

### Дизайн:
- **Чёрный фон** (основной цвет)
- **Минималистичный стиль**
- **Хороший контраст** для читаемости текста
- **Современная типографика**
- **Логотип Asakusa Lab** — предоставить в формате PNG/SVG

### Производительность:
- Быстрая загрузка
- Оптимизированные изображения
- Минимальное потребление ресурсов

### SEO и доступность:
- Семантическая вёрстка (header, main, section, article и т.д.)
- alt-текст для всех изображений
- Доступность для скринридеров (где необходимо)

## 12. Примечания разработчику

- Сайт должен быть **самодостаточным и полным** — нет необходимости в CMS для первой версии (содержимое может быть статичным)
- **Код должен быть чистым и структурированным** — для возможного добавления функциональности в будущем
- **Модульная архитектура** — каждый экран должен быть отдельным компонентом для удобства редактирования
- **Документация** — если будут использоваться специальные техники (например, Intersection Observer для lazy-load), добавить комментарии в код
