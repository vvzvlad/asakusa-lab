О нас

Мы — керамическая студия и сообщество керамистов, расположенное в Москве.

Наша студия — это пространство, где профессионалы и любители керамики находят всё необходимое для творчества и развития своих навыков. Мы создали место, объединяющее единомышленников, увлечённых искусством керамики и науки о глазурях.

У нас есть собственная база для производства глазурей, современное оборудование и комфортная атмосфера для работы.

Наше оборудование

Печи для обжига

  • Большая печь: 80 литров
  • Маленькая (быстрая) печь: 10 литров
    • Время обжига: ~10 часов
    • Возможность проводить 2 обжига в день
    • Идеальна для тестирования и прототипирования
  • Контроллеры собственного производства: позволяют программировать процесс обжига и отслеживать реальный график нагрева для отладки

Гончарный круг

  • 2 гончарных круга iMold Professional V3 (последних версий)
  • Характеристика: мощные, тихие, надежные профессиональные аппараты
  • Выбор качества: студия отдает приоритет лучшему оборудованию

Глазури и их компоненты

  • Собственная база для производства глазурей
  • Обширный запас компонентов для глазурей
  • Обратный осмос (для подготовки воды для глазурей для отсутствия оксидов железа)

Оборудование для приготовления и хранения

  • Много места для хранения материалов
  • Большая удобная раковина с ринзером для мытья баночек
  • Фильтры-воронки для переливания
  • Мешалки (собственного производства)

Вытяжка и вентиляция

  • Вытяжка для печи (позволяет работать во время обжига без неприятных запахов)

Покрасочное оборудование

  • Покрасочная камера
  • Несколько краскопультов
  • Компрессор (большой, но тихий) — позволяет красить много изделий повторяемо

Литьё в гипсовые формы (Slip casting)

  • Большая мешалка для шликера (жидкая керамическая масса для литья)
  • Стол для литья
  • База для производства гипсовых форм
  • Полный цикл: от создания формы до литья

3D-печать и цифровое литьё

Несколько 3D-принтеров для:

  • Печати вспомогательных инструментов
  • Печати вырубок
  • Печати штампов
  • Печати индивидуальных маркировочных штампов для керамистов

Технология "Цифровое литьё" (Digital Casting):

  • Процесс: 3D-модель → печать формы на 3D-принтере → создание силиконовой формы → тиражирование гипсовых форм → литьё шликера в формы
  • Позволяет производить неограниченное количество идентичных гипсовых форм на основе одной печатной формы
  • Курсы по цифровому литью проводятся в студии

Вспомогательные пространства

  • Чистая зона: помещение без пыли для работы за ноутбуком и отдыха на диване
  • Фотозона: оборудована лайтбоксом для фотографирования готовых изделий
  • Выставочная зона: небольшая выставочная галерея в коридоре, где керамисты могут выставить свои работы для демонстрации посетителям
